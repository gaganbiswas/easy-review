# Easy Review

### Step 1:

Create a `.env` file and add the following:

```
PORT=
GOOGLE_REVIEW_REDIRECT_LINK=
BUSINESS_NAME=
SMTP_HOST=
SMTP_PORT=
SMTP_USER=
SMTP_PASSWORD=
SEND_TO_EMAIL=
```

### Step 2:

- Install dependencies `npm install`
- Build your frontend `npm run build`

### Step 3:

Start your server `npm start` or start development server `npm run dev`.
